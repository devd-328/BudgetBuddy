## ✨ Features

- **🎙️ Voice Input**: Dictate your expenses naturally built on natively integrated Web Speech API. (e.g. "_Add 250 for Coffee_")
- **📊 Interactive Analytics**: Layered Doughnut and 6-Month Line Charts to aggressively track exactly where your dollars go.
- **🎯 Dynamic Budgets**: Category-based dynamic limits pushing Color-coded bounds (Green -> Yellow -> Red) notifying you approaching edges securely.
- **🤝 IOU Tracker**: Effortlessly track who you Owe and who Owes you decoupled from native balance flows natively.
- **🤖 Smart Assistant**: Built-in rules-based MVP Chatbot extracting data bounds mathematically securely from your own database payload!

## 🛠️ Tech Stack

- **Frontend**: React 18 / Vite / Tailwind CSS
- **Backend & Auth**: Supabase
- **Charts**: Chart.js / React-ChartJS-2
- **Hosting**: Prepared for Vercel

## 🚀 Getting Started

1. Clone the repository.
2. Run `npm install`
3. Rename `.env.example` to `.env.local` and substitute your Supabase parameters securely.
4. Run `npm run dev` to start your dynamic development server securely!
